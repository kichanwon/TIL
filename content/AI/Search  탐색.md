>탐색 (Search)
>컴퓨터가 문제를 자율적으로 해결하기 위해 해(값) 혹은 해(값)에 이르기 위한 경로를 찾아가는 과정

- 탐색은 인공지능적 문제해결에서 주요한 수단
- 해를 찾는 과정의 효율성과 찾은 해의 적합성까지 포함
- 인공지능 시스템에서 적용할 규칙을 선택하는 제어시스템의 행위는 일종의 탐색 과정
- AI의 핵심적 개념
- 절차적(proceduaral)추론과 다름
- 지수적 폭발문제에 큰 약점을 보임

## 인공지능의 대상 문제
- 해법이 이미 확립되어 있는 것은 포함 X
- 시행착오를 거치면서 해를 구하는 탐색(search)을 필요로 하는 문제 취금

인공지능 문제의 특징
- 탐색형 추론 문제
- 연역형 추론 문제
- 초기상태, 목표상태, 연산자로 표현
- 일반적으로 절차적인 해를 구하는 것이 곤란
- 탐색에 의한 접근을 할 경우 고려해야 될 경우의 수가 폭발적으로 증가

문제 풀이 방법
- 세상의 모든 문제를 푸는 방법에는 크게 세가지가 존재
	- 절차에 기반
	- 탐색에 기반
	- 사례에 기반

*AI 에서의 문제 해결 : 탐색형 추론, 시행착오적 방법으로 문제 해결
