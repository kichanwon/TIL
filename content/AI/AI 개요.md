- 인공지능의 범주
1. Humanly Acting System
	- Turing Test
	- 로봇 공학
	- Cog-humanoid robot
2. Humanly Thinking System
	- GPS(General Problem Solver) : Newell, Simon
	- Cognitive Science : 컴퓨터 모델, 인간 심리 비교
3. Rational Acting System
	- 추론
	- 논리, Syllogism
4. Rational Thinking System
	- 뜨거운 물체에서 손을 갑자기 떼는 것 (생각이 아니라 행동)
	- Agent : 목표를 성취하기 위해 행동, 추론을 포함

탐색에 의한 문제 해결
불확실한 지식의 처리 및 추론
학습
계획
지능형 에이전트

- 지식의 특징
1. 일반성 : 상황별 중요성질 (공통적)
2. 사람이 이해할 수 있어야함
3. 오류처리 용이
4. 완전성 보장 불가
5. 문제 범위 축소 : 지식량 줄임
