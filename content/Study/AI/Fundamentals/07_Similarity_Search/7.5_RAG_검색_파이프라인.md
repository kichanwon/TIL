## Learn About
- RAG(Retrieval-Augmented Generation) 검색 파이프라인  
- 임베딩 기반 정보 통합  

---

## Concept Summary
- Query 임베딩 → 문서 임베딩 검색 → 관련 문서 선택 → LLM 생성 모듈 입력  
- 검색과 생성 통합으로 외부 지식 활용 가능  

---

## Key Points
- 검색 모듈: ANN 기반 유사도 검색  
- 생성 모듈: 선택된 문서 참고하여 응답 생성  
- 임베딩 인덱스 업데이트, 배치 처리, 캐싱 중요  

---

## Examples
- 질문: "2025년 한국 합계출산율?" → 관련 통계 문서 검색 → LLM 요약 답변  
- 문서 업데이트 후 인덱스 재생성 적용
