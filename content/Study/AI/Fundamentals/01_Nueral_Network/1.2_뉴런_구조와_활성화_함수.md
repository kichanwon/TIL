## 학습할 내용
- 인공 뉴런의 기본 구조
- 가중치와 편향의 역할
- 활성화 함수의 필요성과 종류

---
## 개념 요약
- **입력**: 여러 입력값 x1, x2, ... 이 뉴런으로 들어온다.
- **가중치(weight)**: 각 입력은 가중치 w1, w2, ... 과 곱해진다.

| 구분     | 선형 결합 (Linear Combination)    | 활성화 함수(Activation Function)        |
| ------ | ----------------------------- | ---------------------------------- |
| **형태** | 선형 변환 (Linear Transformation) | 비선형 변환 (Non-linear Transformation) |
| **수식** | z = ∑(wᵢ·xᵢ) + b              | a = f(z) = f(∑(wᵢ·xᵢ) + b)         |
| **역할** | 입력 특징들의 중요도를 반영하여 종합          | 뉴런 출력의 범위·특성을 결정, 비선형성 부여          |

---
## 핵심 포인트
- **가중치(Weight)** 
    - 입력 신호의 **중요도**를 조절
    - 너무 크거나 작으면 학습 불안정 → 정규화 필요
- **편향(Bias)**    
    - 임계값을 조정해 **출력 위치(시작점)를 이동**
    - 편향이 없으면 모든 함수가 원점을 지남 → 표현력 제한
- **활성화 함수(Activation Function)**
    - 네트워크가 **복잡한 비선형 관계**를 학습 가능하게 만듦
        - Sigmoid: 확률 출력 (0~1), 하지만 **기울기 소실** 문제 존재
        - Tanh: 출력 범위 (-1~1), Sigmoid보다 중심성 유지
        - ReLU: 음수는 0, 양수는 그대로 → **빠른 학습, 단순 계산**
        - Leaky ReLU: ReLU의 죽은 뉴런 문제 해결 (음수도 작은 기울기 반영)
        - Softmax: 여러 클래스 확률 분포로 변환 (출력층에서 사용)

---
## 시각 자료
- 뉴런 다이어그램: 입력 → 가중치·편향 → 합산 → 활성화 함수 → 출력

---
## 적용 예시
- Sigmoid 함수: 0과 1 사이의 값으로 출력되며, 주로 이진 분류에 사용됩니다.
- Tanh 함수: -1과 1 사이의 값으로 출력되며, 0을 중심으로 대칭적인 활성화 함수입니다.
- ReLU 함수: 0 이상은 그대로 출력하고, 0 이하는 0으로 출력하는 비선형 활성화 함수로, 기울기 소실 문제를 완화할 수 있습니다.
- Leaky ReLU 함수: ReLU의 변형으로, 0 이하의 값에 대해 작은 기울기를 부여해 죽은 뉴런 문제를 해결합니다.
- Softmax 함수: 다중 클래스 분류 문제에서 사용되며, 각 클래스에 속할 확률을 계산합니다.
