### 개념 요약
- **뉴런(Perceptron)**: 입력을 받아 가중합 → 활성화 함수 → 출력
- **활성화 함수**: 비선형성 부여 → 복잡한 패턴 학습 가능

### 주요 활성화 함수
- Sigmoid → 출력 [0,1]
- Tanh → 출력 [-1,1]
- ReLU → 음수 0, 양수 유지
- Softmax → 확률 분포

### 시각 자료
- 뉴런 구조 (입력, 가중치, 합산, 활성화 함수, 출력)
- Sigmoid vs ReLU 곡선 비교

### 예시
- 분류 문제 → Softmax
- 딥러닝 일반 은닉층 → ReLU
