## Concept Summary
- **정의**: 신경망은 뉴런(Neuron)들의 집합으로, 정보 처리를 위한 연결 구조를 의미함.
- **구분**: 
    - **생물학적 신경망** → 실제 뉴런들의 화학적·기능적 연결로 구성
    - **인공 신경망(ANN)** → [인공 뉴런](1.2_뉴런_구조와_활성화_함수)(노드)과 [가중치(weight)](1.2_뉴런_구조와_활성화_함수)로 [모델링(노드 간 연결](1.3_계층_구조와_연결_방식))되어 AI 문제 해결에 활용
- **연결 원리**:
    - 생물학적 → 시냅스(축삭–수상돌기 간 연결)를 통해 신호 전달
    - 인공적 → [가중치](1.2_뉴런_구조와_활성화_함수)를 통해 신호의 세기·방향 조절

## Key Points
- **데이터 흐름**
	- 입력(Input) → [가중치·편향 처리](1.2_뉴런_구조와_활성화_함수) → 비선형 활성화 함수 → 출력(Output)  
- 역할
	- 데이터 → 함수 근사 → 모델 예측
- **학습 방식:**
	- 지도학습(Supervised Learning)
	- 비지도학습(Unsupervised Learning)
	- 강화학습(Reinforcement Learning)
- 주요 구성 요소: 
	- [**뉴런(Neuron)**](1.2_뉴런_구조와_활성화_함수): 입력과 가중치의 선형 조합 + 활성화 함수
	- [**층(Layer)**](1.3_계층_구조와_연결_방식): 입력층, 은닉층, 출력층
	- [**가중치(Weights)**](1.2_뉴런_구조와_활성화_함수): 학습을 통해 조정되는 핵심 파라미터
	- [**편향(Bias)**](1.2_뉴런_구조와_활성화_함수): 출력 이동을 조정하는 추가 파라미터

## Visuals
- **신경망 구조**
```
[입력층(Input Layer)]
       │
       ▼
[은닉층(Hidden Layer)]
       │
       ▼
[출력층(Output Layer)]
```

- **뉴런 내부 연산**
```
입력(x1, x2, …, xn)
     │
  가중치(w1, w2, …, wn)
     │
  합산 + 편향(b)
     │
활성화 함수(f)
     │
   출력(y)
```

- **데이터 흐름 요약 표**

| 단계         | 연산/처리           | 설명                    |
| ---------- | --------------- | --------------------- |
| 입력(Input)  | x1, x2, ..., xn | 모델에 들어가는 원시 데이터       |
| 가중치·편향 처리  | z = Σ(x*w) + b  | 각 입력에 가중치를 곱하고 편향 더하기 |
| 활성화 함수     | y = f(z)        | 비선형 함수 적용으로 출력 결정     |
| 출력(Output) | y               | 최종 모델 예측 결과           |

## Examples

| 분야     | 예시               | 설명                               |
| ------ | ---------------- | -------------------------------- |
| 이미지 인식 | MNIST 손글씨 숫자 판별  | 입력 이미지 데이터를 신경망에 넣어 숫자를 예측       |
| 자연어 처리 | 텍스트 분류, 감정 분석    | 문장 데이터를 기반으로 긍정/부정 감정 또는 카테고리 분류 |
| 추천 시스템 | 사용자 선호 기반 아이템 추천 | 사용자 행동 데이터로 상품, 영화, 음악 등을 추천     |
| 음성 인식  | 음성 → 텍스트 변환      | 오디오 신호를 분석해 문장으로 변환              |
| 자율주행   | 차량 주행 환경 인식      | 카메라/센서 데이터를 기반으로 도로 및 장애물 인식     |
