## Concept Summary
- **Encoder**: 입력 문장 임베딩 → Context 벡터 생성  
- **Decoder**: Context 벡터 + 이전 출력 → 다음 단어 생성  
- 다층 구조로 깊은 의미 표현 가능  

---

## Key Points
- Encoder: Self-Attention + Feed Forward Network  
- Decoder: Masked Self-Attention + Encoder-Decoder Attention + FFN  
- Residual Connection, Layer Normalization 적용
