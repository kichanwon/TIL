## Concept Summary
- 선택된 문서 내용과 Query를 LLM에 입력 → 최종 답변 생성

## Key Points
- Context vector + Query 입력 → LLM 출력  
- Beam Search, Sampling 등 생성 전략 적용  
- Retrieval 결과 품질이 생성 성능 직접 영향
