## Learn About
- 한계와 개선 방향  
- 내부 + 외부 지식 통합 시 고려 사항  

---

## Concept Summary
- 내부 지식: 일부 정보 고정, 외부 지식: 최신/도메인별 보강  
- 한계: 인덱스 품질, 모델 hallucination, 속도 문제  
- 개선: Retrieval 품질 향상, Efficient Fine-tuning, 지식 증류  

---

## Key Points
- 내부 지식 + 외부 지식 병행 → 최적 성능  
- Retrieval 모듈 튜닝 중요  
- 최신 정보 반영과 응답 속도 균형 필요  

---

## Visuals
- 내부 + 외부 지식 통합 구조 그림  
- Retrieval → LLM 생성 흐름 표시

## Examples
- 기업 챗봇: 내부 문서 + 최신 뉴스 기반 응답  
- 학술 QA: 논문 임베딩 + 내부 지식 결합
