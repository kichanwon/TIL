## Concept Summary
- RAG, Retrieval-Augmented Generation과 유사  
- 모델 외부 데이터베이스 또는 문서 임베딩 활용 → 최신/특정 지식 보강

## Key Points
- Query → 외부 문서 검색 → LLM 입력  
- 최신 정보 반영 가능, hallucination 감소  
- 외부 지식 품질에 따라 성능 결정
