# Nueral-Network & LLM+RAG

## 1. 신경망의 주요 적용 분야
- **자연어 처리 (NLP)**  
  텍스트 데이터를 분석·이해·생성  
  예) 기계 번역, 감정 분석, 스팸 필터링, 챗봇  
  특징) 시퀀스 데이터 처리(LSTM, Transformer), 임베딩 활용

- **컴퓨터 비전 (CV)**  
  이미지·영상 데이터 인식 및 처리  
  예) 이미지 분류, 객체 탐지(YOLO), 얼굴 인식  
  특징) CNN, ViT 등 고차원 데이터 처리, GPU 자원 소모 큼

- **음성·오디오 처리**  
  음성 및 소리 신호 분석과 생성  
  예) 음성 인식(STT), 음성 합성(TTS), 음악 장르 분류  
  특징) 파형·스펙트로그램 기반 분석, RNN/Transformer 활용

- **시계열·수치 예측**  
  시간 흐름에 따른 데이터 패턴 분석  
  예) 주가, 날씨, 수요 예측  
  특징) 시계열 패턴 학습, 계절성·추세성 반영

- **생성 모델**  
  새로운 데이터 생성  
  예) GAN, Diffusion 기반 이미지 생성, 텍스트 생성(GPT)  
  특징) 창의적 결과물 생성, 학습 안정화 난이도 높음

- **강화 학습 (RL)**  
  보상 기반 의사결정 학습  
  예) 게임 AI(알파고), 로봇 제어, 자율주행  
  특징) 탐색-활용 균형 필요

- **RAG 및 지식 검색 결합형 AI**  
  검색과 생성 결합  
  예) 사내 문서 QA, 법률 질의응답, 논문 검색 요약  
  특징) 검색 단계와 생성 모델 결합으로 정보 정확성 향상

## 2. 핵심 알고리즘 및 기술 개념 요약
- **Gradient Descent**: 손실 함수 최적화를 위한 대표 알고리즘  
- **Backpropagation**: 오차를 층별 전파해 가중치 업데이트  
- **임베딩과 벡터 표현**: 단어·문장·문서의 의미 기반 수치 표현  
- **유사도 검색**: 임베딩 벡터 간 거리 계산과 효율적 검색  
- **LLM**: 대규모 텍스트로 자연어 이해·생성 수행  
- **Transformer**: 병렬처리와 어텐션 기반 핵심 구조  
- **Attention**: 문맥 내 중요한 정보 집중  
- **RAG**: 외부 지식 검색 결합해 정확하고 최신 답변 생성
