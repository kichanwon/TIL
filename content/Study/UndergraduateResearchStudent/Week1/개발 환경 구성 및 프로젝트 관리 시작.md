### Conda
>- Conda는 오픈 소스, 크로스 플랫폼, 언어에 구애받지 않는 패키지 관리자이자 환경 관리 시스템입니다.
>- Python 패키지 관리 문제를 해결하기 위해 개발되었으며, 현재는Python과 R을 위한 인기 있는 패키지 관리자 입니다.



#### 1.가상환경 생성 및 활성화

```bash
# Python 3.11로 DL_basic이란 이름의 새 환경 생성
conda create -n DL_basic python=3.11

# 환경 활성화
conda activate DL_basic

# 현재 환경 확인
conda info --envs
```

#### 2. CUDA Toolkit / PyTorch 설치 (CUDA 지원)
```bash
conda install -c nvidia cuda-toolkit=12.1

conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia
```

#### 3. 데이터 과학 기본 패키지 설치
```bash
# 과학 계산 및 시각화 패키지 (conda-forge 사용)
conda install pandas matplotlib seaborn scikit-learn -c conda-forge

# Jupyter 설치
conda install jupyter ipykernel -c conda-forge
```
